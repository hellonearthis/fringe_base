# WellyFringe 2026: The Viewer (fringe_calendar.html)

The Viewer is a standalone interactive calendar built with standard HTML, CSS, and JavaScript. It does not require a server or complex dependencies—simply open `fringe_calendar.html` in any modern web browser.

## Files Required
To deploy this viewer, you need:
1. **`fringe_calendar.html`** - The main entry point
2. **`fringe_calendar_app.js`** - The application logic
3. **`fringe_style.css`** - The styling
4. **`fringe_data.js`** - The event data (generated by `generate_calendar.js`)

## Features
- **Calendar View**: Monthly grids for February and March 2026
- **Multi-Show Support**: Correctly handles multiple shows per day and different times on different days.
- **Favorite Indicators**: Colored stars ★ on days with favorite shows
- **Interactive Legends**: Click any Genre or Venue to filter the view
- **Cross-Filtering**: Legends only show categories available under current filters
- **Timeline Visualization**: Visual timeline with drag-and-zoom and current time line
- **Jump Navigation**: Click timeline bars to scroll to show descriptions
- **Visual Feedback**: Inactive filters are dimmed for clear state representation

## Usage
Open `fringe_calendar.html` in any modern web browser.

**Note**: Requires internet connection for Google Fonts (Outfit font family).

## Customization
Edit the CSS variables in `:root` to change the color scheme:
```css
:root {
    --bg: #0f172a;        /* Background */
    --card-bg: rgb(30, 41, 59);  /* Card background */
    --primary: #22d3ee;   /* Primary accent (cyan) */
    --accent: #f43f5e;    /* Secondary accent (pink) */
    --text: #f8fafc;      /* Text color */
    --text-dim: #94a3b8;  /* Dimmed text */
}
```
