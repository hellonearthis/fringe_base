# Fringe Calendar Viewer (fringe_calendar.html)

Interactive calendar view for the Fringe Festival 2026.

## Files Required
To deploy this viewer, you need:
1. **`fringe_calendar.html`** - The main HTML file
2. **`fringe_data.js`** - The event data (generated by `generate_calendar.js`)

## Features
- **Calendar View**: Monthly grids for February and March 2026
- **Favorite Indicators**: Colored stars â˜… on days with favorite shows
- **Show Details Panel**: Click a day to see all shows for that date
- **Timeline Visualization**: Visual timeline of show times
- **Click-and-Drag Zoom**: Select a time range to zoom into
- **Jump Navigation**: Click timeline bars to scroll to show descriptions

## Usage
Open `fringe_calendar.html` in any modern web browser.

**Note**: Requires internet connection for Google Fonts (Outfit font family).

## Customization
Edit the CSS variables in `:root` to change the color scheme:
```css
:root {
    --bg: #0f172a;        /* Background */
    --card-bg: rgb(30, 41, 59);  /* Card background */
    --primary: #22d3ee;   /* Primary accent (cyan) */
    --accent: #f43f5e;    /* Secondary accent (pink) */
    --text: #f8fafc;      /* Text color */
    --text-dim: #94a3b8;  /* Dimmed text */
}
```
